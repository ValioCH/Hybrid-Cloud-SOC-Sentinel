Phase 1: Environment Setup & Hardening
OS Deployment: Чиста инсталация на Windows 11 Pro на физически хардуер.

System Hardening: Използване на Chris Titus Tech Windows Utility за премахване на bloatware, оптимизация на сървисите и затваряне на ненужни портове.

Identity & Domain: Закупуване на персонален домейн и интеграция с Microsoft 365/Entra ID за професионално управление на идентичността.

Phase 2: Hybrid Cloud Connectivity (Azure Arc)
Onboarding: Свързване на физическата машина към Azure през Azure Arc.

Troubleshooting Connectivity: Решаване на проблемите с липсващи логове чрез внедряване на Data Collection Endpoint (DCE).

Agent Deployment: Успешна инсталация на Azure Monitor Agent (AMA) за телеметрия в реално време.

Phase 3: SIEM/SOAR Integration (Sentinel)
Provisioning: Активиране на Microsoft Sentinel върху Log Analytics Workspace.

Data Connectors: Свързване на Defender XDR стека (Endpoint, Identity, Cloud Apps).
